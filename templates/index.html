Welcome to mitcruft.org. We provide Kerberos authentication and related services for former (and current) Athena account holders.

To register your current Athena account, you have a few options:

  • Via HTTP authentication: From any computer with Kerberos tickets, authenticate to our server. This might require additional browser configuration.
  • Via remctl: From any computer with Kerberos tickets, run remctl mitcruft.org register. You may need to install the remctl client.
  • Via MIT certificates: Authentication via expired certificates coming soon!
  • Via your non-MIT email address: If you were forwarding mail elsewhere, we can send a token to that address. Also coming soon!
  • Via a password: You can enter a username and password in our online form.

This process will give you a token that can be used later to create an account, once the MITCRUFT.ORG realm is operational. Save this token, and keep it secure. For updates on the service, join the mitcruft-announce mailing list.
